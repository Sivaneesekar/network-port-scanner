# 🔍 Network Port Scanner

A multi-threaded network port scanner built with Python.

## Features
- Scans open TCP ports on any IP address
- Detects running services (HTTP, SSH, SMB, RPC, FTP)
- Banner grabbing for service identification
- Auto-generates professional HTML report
- Fast scanning using multi-threading

## Technologies Used
- Python 3
- Socket Programming
- Threading
- HTML/CSS Report Generation

## How to Run
python scanner.py

Enter IP address when prompted (e.g., 127.0.0.1)

## Output
- Terminal shows open ports with service names
- HTML report auto-saved in same folder
